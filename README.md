# TraceDraw

1. Alle Dateien zu GitHub hochladen
2. Repository öffnen
3. Settings → Pages
4. Deploy from branch aktivieren
5. Fertig

Danach den Link auf dem Handy öffnen und:
- Zum Homescreen hinzufügen
- Kamera erlauben

Features:
- Live Kamera
- Bild Overlay
- Verschieben
- Drehen
- Zoom
- Deckkraft
- Taschenlampe
